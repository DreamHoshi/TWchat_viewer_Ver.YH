# TWchat_viewer_Ver.YH
TalesWeaverのチャット機能を改善するべく作ったログビューアです
------------------------------------------------------------------------------------
　チャットログビューア　chat_viewer_Ver2.exe
------------------------------------------------------------------------------------
■内容物
------------------------------------------------------------------------------------
・chat_viewer_Ver2.exe　-本体<br>
・settings.json　　-設定ファイル　※設定を維持にチェックを入れると作成されます。
・readme.txt　　　 -今読んでるこれ

■主な機能
------------------------------------------------------------------------------------
・TWゲーム内チャットから出力されたチャットログをリアルタイムで読み取る
　　昨今敵を倒した経験値及びルーン経験値が多すぎてログの流れが激しいことに
　　嫌気が差したため作成。
・チャットの横に据えてフィルタをかけてお使いください。
　ログが流れる原因である経験値･ペットの拾得物・ELSOは表示していません。
・クリック透過機能が備わっているので設定すれば移動の阻害もしません。

■動作環境
------------------------------------------------------------------------------------
TWインストール済みのPC
自分のPC（Windows11）では動きました。

■使い方
------------------------------------------------------------------------------------
・まずTWにログインします
　　オプション　ゲームタブ、チャットログをONにします。
・chat_viewer.exeを開きます
　　テイルズウィーバーのインストール先がデフォルトならフォルダ選択はそのままです。
　　デフォルトのTWインストール先→C:\Nexon\TalesWeaver
・読み込み開始を押します
　　右の停止中だった文字が赤色で動作中となったはず。これでログを読み始めます。

■各種設定/機能
------------------------------------------------------------------------------------
長くなってしまったので割愛、本体付属のりーどみーで読んでください。
